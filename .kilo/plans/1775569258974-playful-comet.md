## Task 1: Create `TeacherAnnouncementInvite.tsx`

**Purpose:** Compact invite management component for teachers, integrated into announcement area

**Location:** `src/modules/user/ui/components/TeacherAnnouncementInvite.tsx`

**Features:**

- Badge showing count of pending join requests
- Click opens a dialog with:
  - List of pending requests with approve/reject buttons
  - Create new invite code form
  - List of active invite codes with copy link functionality
- Uses existing `TeacherInviteLink.tsx` logic but in dialog format

**Props:**

```typescript
interface TeacherAnnouncementInviteProps {
  classId: string;
  organizationSlug: string;
}
```

**Implementation:**

- Create dialog state (`const [isOpen, setIsOpen] = useState(false)`)
- Fetch pending requests count for badge display
- Wrap existing invite code management UI in a Dialog component
- Use existing TRPC procedures: `admin.getManyJoinRequests`, `admin.createInviteCode`, `admin.processJoinRequest`, `admin.deleteInviteCode`

## Task 3: Create `StudentJoinStatus.tsx`

**Purpose:** Display-only component showing student's join request status for a class

**Location:** `src/modules/user/ui/components/StudentJoinStatus.tsx`

**Features:**

- Check student's join request status via `trpc.user.getJoinRequestStatus`
- Display status-appropriate UI:
  - **No request:** Show nothing or minimal "Not a member" indicator
  - **Pending:** Yellow/warning state with message "Your request to join is pending approval"
  - **Approved:** Green/success state with message "You are a member" + link to class
  - **Rejected:** Red/error state with message "Your request was rejected"

**Props:**

```typescript
interface StudentJoinStatusProps {
  classId: string;
  organizationSlug: string;
}
```

**Implementation:**

- Server component that fetches status via TRPC
- Render appropriate UI based on status
- Include "Request to Join" flow if no request exists (uses existing `/join/[slug]` page)
