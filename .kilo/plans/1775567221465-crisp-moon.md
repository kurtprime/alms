# Plan: Image Cropping/Editing for Class Header Upload

## Context

Currently, when a teacher uploads a header image, the full image is used as-is. Teachers need the ability to:

- Crop the image to fit the header area
- Zoom in/out to select the best portion
- Rotate the image if needed
- Preview how the image will look as a background

## Approach

**Using `react-image-crop`** (already downloaded in the project) with additional zoom functionality via canvas scaling.

The header image aspect ratio is approximately **3:1** (h-32 md:h-54, wide banner style).

## Implementation Steps

### Step 1: Verify react-image-crop Installation

Check if `react-image-crop` is already in package.json. If not, it will be added.

### Step 2: Update `ClassHeaderImage` Component

**File:** `src/modules/user/ui/components/ClassHeaderImage/index.tsx`

**State Additions:**

- `selectedFile: File | null` - The original file selected by user
- `croppedImageUrl: string | null` - The cropped/edited image as a data URL
- `zoom: number` - Zoom level (1 = 100%, default 1)
- `rotation: number` - Rotation in degrees (default 0)

**UI Flow:**

1. User clicks to select image → file input opens
2. After selecting, show the ImageCropper modal instead of directly uploading
3. ImageCropper shows:
   - The image with crop overlay (3:1 aspect ratio)
   - Zoom slider (0.5x to 3x)
   - Rotate left/right buttons (90° increments)
   - Preview of final result
   - Cancel and Apply buttons
4. On Apply:
   - Use canvas to render the cropped/transformed image
   - Convert to blob/file
   - Upload via existing `startUpload("classHeaderUploader")`

**Component Structure:**

```
ClassHeaderImage
├── Header display (existing)
│   └── <div className="absolute inset-0 bg-cover..." />
└── ResponsiveDialog (edit modal)
    └── ImageCropper
        ├── ReactCrop (with aspect={3/1})
        ├── Zoom slider
        ├── Rotate buttons
        ├── Preview thumbnail
        └── Action buttons
```

### Step 3: Create `ImageCropper` Sub-Component

**File:** `src/modules/user/ui/components/ClassHeaderImage/ImageCropper.tsx`

A separate component for the editing interface containing:

- `ReactCrop` from `react-image-crop` with `aspect={3}` (width/height ratio)
- Slider for zoom (controlled via CSS transform on the cropper)
- Rotate buttons (maintain state, apply transform)
- "Apply" button that:
  1. Creates an off-screen canvas
  2. Draws the image with crop coordinates, zoom, and rotation
  3. Exports as blob
  4. Calls `onCropComplete(blob)` callback

**Props:**

```tsx
interface ImageCropperProps {
  imageFile: File;
  aspect?: number; // 3 for 3:1 ratio
  onCropComplete: (croppedBlob: Blob) => void;
  onCancel: () => void;
}
```
