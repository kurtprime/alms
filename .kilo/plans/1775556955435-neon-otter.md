# Plan: Add Icons and Adjust Tab List Width in ClassIdView

## Goal

Improve the vertical tabs sidebar by:

1. Adding lucide icons to each tab
2. Making the tab list narrower with more padding
3. Ensuring content area takes remaining width

## Changes to `src/modules/user/ui/Views/ClassIdView.tsx`

### 1. Import Lucide Icons

Add imports for the icons matching each tab:

- `Bell` for Announcements
- `LayoutDashboard` for Overview
- `Users` for Students
- `BookOpen` for Gradebook

### 2. Modify TabsList Class

Change from `md:flex-1` to a fixed width with padding:

- Replace `md:flex-1` with `md:w-56 md:p-4`
- This gives ~224px width for the sidebar with comfortable padding

### 3. Update TabsTrigger Components

For each TabsTrigger, add:

- An icon component before the text label
- Adjust text styling for icon + text layout

**Announcements tab:**

```tsx
<TabsTrigger value="announcement" className="...">
  <Bell className="md:mr-2 h-4 w-4" />
  <span>Announcements</span>
</TabsTrigger>
```

**Overview tab:**

```tsx
<TabsTrigger value="overview" className="...">
  <LayoutDashboard className="md:mr-2 h-4 w-4" />
  <span>Overview</span>
</TabsTrigger>
```

**Students tab:**

```tsx
<TabsTrigger value="students" className="...">
  <Users className="md:mr-2 h-4 w-4" />
  <span>Students</span>
</TabsTrigger>
```

**Gradebook tab (teacher only):**

```tsx
<TabsTrigger value="gradebook" className="...">
  <BookOpen className="md:mr-2 h-4 w-4" />
  <span>Gradebook</span>
</TabsTrigger>
```

### 4. Update TabsTrigger Classes

Add `md:flex-row` to ensure icon appears before text on desktop, and adjust padding:

- Add `md:flex-row md:items-center` to the className
- The existing `md:w-full md:justify-start` will handle alignment

## Summary of Class Changes

| Element        | Old             | New                         |
| -------------- | --------------- | --------------------------- |
| TabsList       | `md:flex-1`     | `md:w-56 md:p-4`            |
| TabsTrigger    | text only       | icon + text with `md:mr-2`  |
| Tabs className | `flex flex-row` | unchanged (already correct) |
