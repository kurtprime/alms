# Organization Invite Link System

## Overview

Implement per-organization invite links with:

- URL format: `/join/[orgSlug]?code=XYZ`
- Single-use invite codes
- Admin approval required before joining

## Database Schema

### New Table: `orgInvitation`

File: `src/db/schemas/organization-schema.ts`

```typescript
export const orgInvitation = pgTable('org_invitation', {
  id: varchar('id', { length: 255 }).primaryKey(),
  organizationId: varchar('organization_id', { length: 255 }).notNull(),
  code: varchar('code', { length: 32 }).notNull().unique(),
  role: organizationMemberRole('role').default('student').notNull(),
  createdBy: varchar('created_by', { length: 255 }).notNull(),
  usedBy: varchar('used_by', { length: 255 }),
  usedAt: timestamp('used_at'),
  expiresAt: timestamp('expires_at').notNull(),
  status: text('status').default('pending').notNull(), // pending, used, expired, revoked
});
```

### New Table: `joinRequest`

File: `src/db/schemas/organization-schema.ts`

```typescript
export const joinRequest = pgTable('join_request', {
  id: varchar('id', { length: 255 }).primaryKey(),
  organizationId: varchar('organization_id', { length: 255 }).notNull(),
  userId: varchar('user_id', { length: 255 }).notNull(),
  invitationId: varchar('invitation_id', { length: 255 }).notNull(),
  status: text('status').default('pending').notNull(), // pending, approved, rejected
  createdAt: timestamp('created_at').notNull().defaultNow(),
  processedAt: timestamp('processed_at'),
  processedBy: varchar('processed_by', { length: 255 }),
});
```

## tRPC Procedures

### Admin Procedures (`src/modules/admin/server/procedures/section.ts`)

1. **`section.createInvitation`**
   - Input: `{ organizationId, role }`
   - Generates unique 8-char code
   - Sets expiration (7 days default)
   - Returns: `{ code, url, expiresAt }`

2. **`section.listInvitations`**
   - Input: `{ organizationId }`
   - Returns: List of all invitations with status

3. **`section.revokeInvitation`**
   - Input: `{ invitationId }`
   - Sets status to "revoked"

4. **`section.approveJoinRequest`**
   - Input: `{ requestId }`
   - Creates member record, updates invitation to "used"

5. **`section.rejectJoinRequest`**
   - Input: `{ requestId }`
   - Updates request status to "rejected"

### User Procedures (`src/modules/user/server/procedures/class.ts`)

1. **`joinClassByCode`**
   - Input: `{ organizationSlug, code }`
   - Validates code exists and not used/expired
   - Creates pending join request
   - Returns: `{ requestId, status }`

2. **`getJoinRequestStatus`**
   - Input: `{ requestId }`
   - Returns current status of join request
