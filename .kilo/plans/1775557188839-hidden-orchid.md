# Plan: Increase Tab Sizes

## Goal

Make the class tabs (Announcements, Overview, Students, Gradebook) larger with bigger icons, text, padding, and margins.

## Current State

- Icons: `h-4 w-4` (16px)
- Text: `text-sm`
- Padding: `px-4`
- Icon-text spacing: `mr-2`
- TabsList width: `w-56` with `p-4` padding

## Proposed Changes (Bigger Sizes)

### 1. Increase Icon Size

Change icons from `h-4 w-4` to `h-6 w-6` (24px)

### 2. Increase Text Size

Add `text-lg` to TabsTrigger for larger text

### 3. Increase Horizontal Padding

Change `px-4` to `px-8` (32px horizontal padding)

### 4. Increase Icon-Text Spacing

Change `mr-2` to `mr-4` (16px spacing)

### 5. Increase TabsList Padding

Change TabsList `p-4` to `p-6` on desktop

## Implementation

### File: `src/modules/user/ui/Views/ClassIdView.tsx`

**TabsList:**

```tsx
className =
  'h-12 w-full justify-start rounded-none bg-transparent p-4 md:flex-col md:h-auto md:w-56 md:p-6 md:justify-start md:rounded-none md:bg-transparent';
```

**Each TabsTrigger:**

```tsx
<Bell className="mr-4 h-6 w-6" />
```

```tsx
<LayoutDashboard className="mr-4 h-6 w-6" />
```

```tsx
<Users className="mr-4 h-6 w-6" />
```

```tsx
<BookOpen className="mr-4 h-6 w-6" />
```

And add `text-lg` to the className along with increased padding `px-8`.
